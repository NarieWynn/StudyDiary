## 1.1 Hello World

```rust
fn main(){
	println!("Hello, World!");
}
```
Note: println! calls a Rust macro, it it not a normal function (Discuss later for now using ! is calling macro).
## Compile and Execution 
- Compile Rust program using "rustc" command 
```
$ rustc main.rs
```
Note: Similar to C or C++ (gcc or clang). After compiling Rust output a binary executable.
- Run file using 
```
$ ./main 
```
**Note:** Rust is an ahead-of-time compiled language => after compiling can run file without install rust (Run directly on CPU)
## 1.2 What is Cargo
- Cargo is a Rust build system (similar to [[CMake#^cmake-def|CMake]])
- Cargo building your code, downloading libraries your code depends on version
- Check whether cargo is installed or not 
```
$ cargo --version
```

## Creating a project with cargo
```
$ cargo new hello_cargo //create a new directory call hello_cargo
$ cd hello_cargo // go into hello_cargo directory
```
