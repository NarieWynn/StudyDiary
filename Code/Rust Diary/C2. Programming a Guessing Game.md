## Processing a Guess
Filename: src/main.rs
```rust
use std::io;

fn main(){
	println!("Guess the number!");
	
	println!("Please input your guess.");
	
	let mut guess = String::new();
	
	io::stdin()
		.read_line(&mut guess)
		.expect("Failed to read line");
	
	println!("You guessed: {guess}");
}
```
- use std::io;
This is the standard (std:) library io to receive input and output result  using this io input/output library
# By default, Rust has a set of items defined in the standard library, this set called the prelude (update the list of prelude later in chapter 12). If a type you want to use isnt in the prelude you has to "use" statement

## Storing Values with Variables
- Next, create a variable to store user input
```rust
let mut guess = String::new();
```
we use "let" statement to create the variable
```rust
let apple = 5;
```
this line creates a new variable named apples and binds it to the value 5. In Rust, variables are immutable by default, meaning once we give the variable a value, the value  won't change. See more in chapter 3
=>To make a variable mutable, add "mut" before the variable name
```rust
let apples = 5; // immutable
let mut bananas = 5; // mutable
```
